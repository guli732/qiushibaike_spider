# 糗事百科段子爬虫
## 简介
- 通过使用scrapy框架创建Spider爬虫，爬取糗事百科段子，代码量很少，适合初学者跑通scrapy流程。
- 仅作为学习使用
## 爬取字段
- `author`：作者
- `content`：内容
